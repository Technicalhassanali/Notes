HTML

HTML stands for HyperText Markup Language.HTML is the standard markup language for documents designed to be displayed in a web browser.

Syntax : 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    
</body>
</html>

<head> </head> | Head Tag

<head>tag contains metadata (document title, character set, styles, links, scripts), specific information about the web page that is not displayed to the user. Metadata provides browsers and search engines with technical information about the web page.

<head></head>

<title> </title> | Title Tag

<title>tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.It is most important for search engine optimization(SEO).

<title>Title Page </title>

<body> <body> | Body Tag

<body> ag defines the main content of an HTML document which displays on the browser. It can contain text content, paragraphs, headings, images, tables, links, videos, etc. 

<body> <body>

bgColor="" attribute is change the background color of text.

width="" attribute is the set the width(length) of element.

height="" attribute is used for set the height of element.

<a> </a> | HyperLink Tag

this tag name is hyperlink, and it is used to link the other document or file.<a> tag defines a hyperlink, which is used to link from one page to another.

//target = "_blank"," _parent","_self"," _top";

<img> </img> | Image Tag

<img> tag is used to embed an image in an HTML page.

Attributes :

src Specifies the path to the image.

alt name of image,that help in seo.

<img src="img_girl.jpg" alt="Girl in a jacket"/>

<table> </table> | Table Tag

<table> tag is used make the table in the doucment.

border="1" attribute is used for the border of table.

<caption>tag defines a table caption.caption is the display name of table.

<tbody>tag is used to group the body content in an HTML table.

<tr>tag is used for table row.

<th>tag is used for table height.

<td>tag is used for table data.

<tfoot>tag is used to group footer content in an HTML table.

<table border="1" >
    <caption>Table Name</caption>
    <tbody>
        <tr>
            <th>Heading 1</th>
            <th>Heading 2</th>
        </tr>
        <tr>
            <td>column 1</td>
            <td>column 2</td>
        </tr>
    </tbody>
        <tfoot>  
        <tr>
            <td></td>
            <td></td>
        </tr>
    </tfoot>
</table>

colspan = "2"The value of the colspan attribute represents the number of columns to span.

rowspan = "2"The value of the rowspan attribute represents the number of rows to span.

face="" attribute is used for change the font family of text.

size="" attribute is used increase the size of text.
